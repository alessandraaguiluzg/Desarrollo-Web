Datos Generales sobre la Materia

Este curso es el primero que recibo de desarrollo web y cubre los fundamentos sobre HTML, CSS y JavaScript, que son los elementos esenciales de cualquier aplicación web.

A través del dominio de los elementos de HTML y el uso de los elementos de estilo en CSS los estudiantes hemos adquirido la habilidad de diseñar sitios web atractivos. Próximamente con el uso de los conceptos básicos y sintaxis de JavaScript, los estudiantes podremos crear aplicaciones interactivas de front-end.
